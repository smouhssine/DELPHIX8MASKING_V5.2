# DELPHIX_MASKING_ALGOS

To extend your delphix engine with the available algorithms from this repos, please follow those steps

1/ Download txt algorithm files <br />
2/ Login to your masking engine using and admin account <br />
3/ Click Add Algorithm at the top right of the Algorithm tab <br />
4/ Select "Secure lookup algorithm" for algorithm type <br />
5/ Complete the "Algorithm name" and "Description" <br />
6/ Load the lookupfile you've downloaded from this repo.
